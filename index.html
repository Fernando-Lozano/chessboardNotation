Master The Chessboard

Learn to memorize the coordinates of the chessboard in a fun way. Instead of focusing on learning each individual square, which is 64 squares, focus on learning the rows and columns, which is only 16 things to memorize. First, memorize the rows from every angle. Then, memorize the columns from any angle. And finally, memorize the squares from any angle.
